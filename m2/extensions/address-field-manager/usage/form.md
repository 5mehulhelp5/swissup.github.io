# New Address Field Form

Address Field Manager interface is available at _"Swissup > Address Field Manager"_
menu.

To create new address field, press `Add New Field` button.

Available field types:

 -  Text Field
 -  Text Area
 -  Date
 -  Yes/No
 -  Multiple Select
 -  Dropdown
