# New Customer Field Form

Customer Field Manager interface is available at _"Swissup > Customer Field Manager"_
menu.

To create new customer field, press `Add New Field` button.

Available field types:

 -  Text Field
 -  Text Area
 -  Date
 -  Yes/No
 -  Multiple Select
 -  Dropdown
