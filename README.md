### Installation

```bash
git clone git@github.com:swissup/swissup.github.io.git && cd swissup.github.io

# Install bundler is you don't have it
gem install bundler

# Install dependencies
bundle install && bower install && npm install
```

### Runing Locally

```bash
gulp

# OR

bundle exec jekyll serve
```
