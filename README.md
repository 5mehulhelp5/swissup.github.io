### Installation

```bash
git clone git@github.com:swissup/swissup.github.io.git && cd swissup.github.io

# Install bundler is you don't have it
gem install bundler

# Install dependencies
bundle install
```

### Updating

```bash
bundle update
```

### Runing Locally

```bash
bundle exec jekyll serve
```
